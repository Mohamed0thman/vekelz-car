## vekelz-car

## Usage

by

- installing app's dependency
  ```shell
    npm install
  ```
- start dev server by using
  ```shell
    npm run dev
  ```

## Features

- (Home Page, Bookin Page)
- Interactive chart using chart.js
- Full responsive web app
- dark mode
